    • Tags 2.0 Icons Reference

    Tags 2.0 introduces new features, previously not available to Mattertags. These features include the ability to select an icon from Matterport’s font library.

    We have provided this reference to be used to visualize the icons that are returned from Model API.

    Icons can now be queried from Model API (Try It!):

    query getTags($modelId:ID!) {
	model(id: $modelId) {
		mattertags {
			id
			label
			description
			icon
			color
			position { x y z }
			stemNormal { x y z }
		}
	}
}
