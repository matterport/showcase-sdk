moveToMode

  • moveToMode(mode: ModeEnum): Promise<ModeEnum>

  • Change the viewing mode in 3D Showcase.

mpSdk.moveToMode(mode)
  .then(function(nextMode){
    // moveToMode complete.
  })
  .catch(function(error){
    // moveToMode error.
  });
    deprecated

    Parameters

    • mode: ModeEnum

      The mode.

    Returns Promise<ModeEnum>

    A promise that resolves with the new mode once it has been reached.

moveToPano

  • moveToPano(options: MoveToPanoOptions): Promise<string>

  • Move to a new panorama (a new scan position in Inside View or a new 360º View).

    Errors

    • Fails if the provided UUID does not exist in the list of panos.
    • Fails if the rotation or transition is specified incorrectly.
    const panoId = '1';
const rotation = {0,0,0};
const transition = mpSdk.Sweep.TransitionType.INSTANT;

mpSdk.moveToPano({
    pano: panoId,
    rotation: rotation,
    transition: transition
  })
  .then(function(sweepId){
    // moveToPano successful.
    console.log('Arrived at sweep ' + sweepId);
  })
  .catch(function(error){
    // Error with moveToPano command
  });
    deprecated

    Parameters

    • options: MoveToPanoOptions

      the input.

    Returns Promise<string>

    A promise that resolves with the pano sid when the destination pano has been reached.

takeScreenshot

  • takeScreenshot(resolution?: Camera.Resolution, visibility?: Camera.Visibility): Promise<string>

  • Takes a screenshot (JPEG) of the user’s current view.

    const resolution = {
  width: 600,
  height: 800
};

const visibility = {
  mattertags: false,
  sweeps: true
};

mpSdk.takeScreenShot(resolution, visibility)
  .then(function (screenShotUrl) {
    // set src of an img element
    img.src = screenShotUrl
});
    deprecated

    Errors

    Warns if the resolution is 0 or negative.

    Notes

    The URL only exists (temporarily) on the client’s browser. It is not on Matterport servers and does not persist between user sessions.

    Parameters

    • Optional resolution: Camera.Resolution

      The desired resolution for the screenshot. For example: {width: 1920, height: 1080} If no resolution is specified, then the resolution of the size of Showcase (the current window or the iframe embed) is used. Maximum 4096 x 4096.

    • Optional visibility: Camera.Visibility

      Toggles certain scene objects such as Mattertag Posts and pano markers. If no visibility object is specified, then all scene objects are hidden.

    Returns Promise<string>

    A promise that resolves with the screenshot url.
