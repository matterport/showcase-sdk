MoveToOptions

MoveToOptions: object

rotation.x: is the amount the camera will rotate up/down, in the range between [-90…90] with -90 being straight down and 90 being straight up, 45 would be looking up at a 45 degree angle., -45 down etc.. rotation.y: is the amount the camera rotate around horizontally, between [-360…0…360], negative values to rotate to the left, positive to rotate to the right.

Note: The rotation that Sweep.moveTo uses for input is the same rotation that will get returned from the Camera.getPose command.

const pose = await mpSdk.Camera.getPose();
console.log(pose.rotation);

With that combo you can get your view lined up with the normal camera, call getPose, and use that as input to the moveTo command.

Type declaration

  • Optional rotation?: Vector2
  • Optional transition?: Sweep.TransitionType

moveTo

  • moveTo(sweep: string, options: Sweep.MoveToOptions): Promise<string>

  • Move to a sweep.

    const sweepId = '1';
const rotation = { 30, -45 };
const transition = mpSdk.Sweep.Transition.INSTANT;

mpSdk.Sweep.moveTo(sweepId, {
    rotation: rotation,
    transition: transition
  })
  .then(function(sweepId){
    // Move successful.
    console.log('Arrived at sweep ' + sweepId);
  })
  .catch(function(error){
    // Error with moveTo command
  });

    Parameters

    • sweep: string
    • options: Sweep.MoveToOptions

    Returns Promise<string>

    A promise that will return the destination sweep.
