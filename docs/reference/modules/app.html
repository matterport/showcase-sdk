getLoadTimes

  • getLoadTimes(): Promise<object>

  • Get the time that each app phase has loaded

    mpSdk.App.getLoadTimes()
  .then(function(loadTimes) {
     if (loadTimes[mpSdk.App.Phase.STARTING] !== null) {
       const timeToLoad = loadTimes[mpSdk.App.Phase.STARTING] - loadTimes[mpSdk.App.Phase.LOADING];
       console.log('It took ', timeToLoad, ' milliseconds to see the model after the play button was clicked');
     }
     else {
       console.log('The model is not visible yet.');
     }
  });

    Returns Promise<object>

getState

  • getState(): Promise<App.AppState>

  • get the current running Application, and its Phase.

    mpSdk.App.getState()
  .then(function(state) {
    console.log('Current Application: ' + state.application);
    console.log('Current Application Phase: ' + state.phase);
  })
  .catch(function(error) {
    // App data retrieval error.
  });

    Returns Promise<App.AppState>
