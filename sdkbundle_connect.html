    Connecting to the SDK

    Add the applicationKey to your bundle iframe’s URL parameters

     <iframe id="showcase" width="740" height="480" src="/bundle/showcase.html?m=[MODEL_SID]&applicationKey=[APPLICATION_KEY]" frameborder=”0” allowfullscreen allow="vr"'></iframe>

    Connect to the SDK

    Connect using the simplified syntax connect(window: Window)

    const showcase = document.getElementById('showcase');
showcase.addEventListener('load', async function() {
  const sdk = await showcase.contentWindow.MP_SDK.connect(showcase.contentWindow);
  console.log('Hello Bundle SDK', sdk);
});

    Connect using an OAuth Token

      const sdk = await showcase.contentWindow.MP_SDK.connect(showcase.contentWindow, {
    auth: '[access_token]',
  });
